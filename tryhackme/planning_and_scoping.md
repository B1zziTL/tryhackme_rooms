**Rooms:** x | x | Governance \& Regulation



**Theory:**

* *Governance* = directing an organization to achieve its objectives
* *Regulation* = rule/law enforced by governing body to ensure compliance
* *Compliance* = the state of adhering to laws, regulations and standards



Laws/Regulations:

* *GDPR* = General Data Protection Regulation
* *HIPAA* = Health Insurance Portability and Accountability Act
* *PCI-DSS* = Payment Card Industry Data Security Standard
* *GLBA* = Gramm-Leach-Bliley Act
* *NIST 800-53* = Compliance Best Practices

  * Discover and classify sensitive data
  * Map data and permissions
  * Manage access control
  * Monitor data, file activity and user behaviour



Preparing a Password Policy:

1. password requirements
2. password usage guidelines
3. password storage \& transmission guidelines
4. password change \& reset guidelines
5. communicating the policy
6. monitoring compliance
