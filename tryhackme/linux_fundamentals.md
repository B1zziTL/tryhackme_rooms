**Commands:**



```$ echo``` = output text in terminal

```$ whoami``` = current logged user

```$ ls``` = list content in current folder

```$ cd``` = change directory

```$ cat``` = concatenate / read file content

```$ pwd``` = current full path



```$ find -name name\_of\_file.txt``` = looking for specific file known by name

```$ find -name \*.txt``` = looking for all .txt files

```$ grep "x.y.z.w" access.log``` = search entire content for any values

&#x20;```-R``` = search recursively



```\&``` = run commands in the background

```\&\&``` = run 2nd command if 1st successful

```>``` = redirect content (rewrite)

```>>``` = redirect content (append)
